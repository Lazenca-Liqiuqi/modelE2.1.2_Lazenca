# HYCOM

HYCOM is an isopycnal model based on MICOM that has a hybrid mixed layer scheme near the surface. This has been fully coupled to the modelE atmosphere. For more details contact [ssun@giss.nasa.gov](mailto:ssun@giss.nasa.gov).
HYCOM是基于MICOM的等密度面模型，近表面具有混合层方案。该模型已与modelE大气完全耦合。更多详情请联系[ssun@giss.nasa.gov](mailto:ssun@giss.nasa.gov)。（原文拼写：isopyncal）
