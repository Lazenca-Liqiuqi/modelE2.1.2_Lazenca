# Ocean Tracers (Russell Ocean only) / 海洋示踪物（仅限Russell海洋）

These are turned on using `#define TRACERS_OCEAN`.
这些功能通过`#define TRACERS_OCEAN`开启。

If no other directives are set, then the tracer code will only be used in the ocean, but if the atmospheric water tracers are also turned on, then tracers can be used throughout the whole system.
如果未设置其他指令，则示踪物代码将仅在海洋中使用，但如果大气水示踪物也开启，则示踪物可以在整个系统中使用。
